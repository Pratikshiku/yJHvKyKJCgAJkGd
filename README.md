## Programming Training Examination System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/56074afb757f4617b0b3867b49b5bd3b/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Question Bank Resources
5. User Communication
6. Exam Paper List
7. Training Notifications
8. Message Feedback
9. Personal Center
10. My Publications
11. Exam Records
12. Wrong Answer Book
13. My Collections
14. Change Password
15. Practice Statistics
16. Score Statistics
17. My Message Board
18. My Collection Management
19. My Exam Management


**Administrator Features:**
1. Login
2. Personal Center
3. User Management
4. Question Bank Resource Management
5. Practice Statistics Management
6. Score Statistics Management
7. Score Statistics
8. Exam Paper Management
9. Message Board Management
10. Question Management
11. User Communication
12. Training Notifications
13. Carousel Management
14. Exam Records
15. Wrong Answer Book
